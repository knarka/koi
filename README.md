koi is a tiny go text editor.

## Keybindings
### General
| key | action |
|-----|--------|
| C-c | quit   |

### Movement
| key          | action                |
|--------------|-----------------------|
| C-k or up    | move the cursor up    |
| C-j or down  | move the cursor down  |
| C-h or left  | move the cursor left  |
| C-l or right | move the cursor right |
